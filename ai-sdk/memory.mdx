## Custom Memory Implementation

Implement the `Store` interface for custom backends:

```go
type RedisMemoryStore struct {
    client *redis.Client
}

func (s *RedisMemoryStore) GetConversation(
    ctx context.Context,
    filter memory.Filter,
) (types.Conversation, error) {
    // Retrieve from Redis
    key := fmt.Sprintf("conversation:%s", filter.SessionID)
    data, err := s.client.Get(ctx, key).Bytes()
    if err != nil {
        // Create new conversation if not found
        return types.Conversation{
            ID:        uuid.New().String(),
            SessionID: filter.SessionID,
            Messages:  []types.Message{},
            Status:    types.StatusActive,
        }, nil
    }
    
    var conv types.Conversation
    json.Unmarshal(data, &conv)
    
    // Apply limit if specified
    if filter.Limit > 0 && len(conv.Messages) > filter.Limit {
        conv.Messages = conv.Messages[len(conv.Messages)-filter.Limit:]
    }
    
    return conv, nil
}

func (s *RedisMemoryStore) SaveConversation(
    ctx context.Context,
    conversation types.Conversation,
) error {
    // Save to Redis
    key := fmt.Sprintf("conversation:%s", conversation.SessionID)
    data, _ := json.Marshal(conversation)
    return s.client.Set(ctx, key, data, 24*time.Hour).Err()
}
```

Use custom store:

```go
redisStore := &RedisMemoryStore{client: redisClient}

ag, _ := agent.New(
    agent.WithModel(model),
    agent.WithMemory(redisStore),
)
```

## Memory Hooks

From `pkg/ai-sdk/agent/agent.go:51-56`:

Monitor memory operations with hooks:

```go
ag, _ := agent.New(
    agent.WithModel(model),
    agent.WithMemory(memoryStore),
    agent.WithHooks(agent.Hooks{
        OnBeforeMemoryRetrieve: func(ctx context.Context, filter memory.Filter) {
            log.Printf("Retrieving conversation for session: %s", filter.SessionID)
        },
        OnMemoryRetrieved: func(ctx context.Context, filter memory.Filter, conv types.Conversation) {
            log.Printf("Retrieved %d messages", len(conv.Messages))
        },
        OnBeforeMemorySave: func(ctx context.Context, conv types.Conversation) {
            log.Printf("Saving conversation: %s", conv.ID)
        },
        OnMemorySaved: func(ctx context.Context, conv types.Conversation) {
            log.Printf("Saved successfully")
        },
    }),
)
```

## Conversation History Limit

From `pkg/ai-sdk/agent/options.go:61-65`:

Limit messages retrieved from memory:

```go
ag, _ := agent.New(
    agent.WithModel(model),
    agent.WithMemory(memoryStore),
    agent.WithConversationHistoryLimit(50), // Last 50 messages only
)
```

This prevents:
- Exceeding model context windows
- High token costs
- Slow API calls

Set to `0` for unlimited (all messages).

## Best Practices

1. **Use SessionID consistently** for conversation continuity
2. **Set conversation history limits** to manage context window
3. **Index your database** properly for performance
4. **Handle conversation status** (active, interrupted, completed)
5. **Store metadata** for tracking and analytics
6. **Implement pagination** for large conversations
7. **Clean up old conversations** periodically
8. **Use memory hooks** for logging and monitoring
9. **Consider TTL** for temporary conversations
10. **Validate conversation ID** before saving
