Flowbaker provides 46+ integrations to connect your workflows with external services, databases, AI models, and utilities. Each integration is designed to be self-hosted and secure, with no data leaving your infrastructure.

## How Integrations Work

Integrations in Flowbaker consist of:

- **Credentials**: Secure authentication configuration (API keys, OAuth, connection strings)
- **Actions**: Operations you can perform (send message, query database, generate text)
- **Triggers**: Events that start workflows (new message, scheduled time, webhook)

## Getting Started

To use an integration:

1. **Configure Credentials**: Set up authentication in the Flowbaker UI
2. **Add to Workflow**: Drag the integration node into your workflow
3. **Configure Action**: Choose the action and set parameters
4. **Connect Nodes**: Link the integration to other workflow nodes
5. **Test**: Run your workflow to verify the integration works

## Security

All integration credentials are:

- Stored encrypted in your database
- Never sent to external services except when making authenticated requests
- Scoped to your Flowbaker instance
- Protected by your authentication layer
